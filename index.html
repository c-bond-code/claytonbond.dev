About This Project

This is a simple implementation of a Bootstrap Front-End hosted on Apache. It securely interacts with SHA256 to a MySQL backend, hosted through a Docker container.

Allows any user with appropriate permissions (Teachers) to view, edit, update, create and delete users. Anyone without permission (Students) can still view users, but edit options will be hidden to them.

Note

There are several thousand students and a few hundred professors. Feel free to interact and mess around however you see fit. This is public, so there may be some weird edits by other users. If you notice things are messed up, or you can't login, just reach out and I can reset the database.
